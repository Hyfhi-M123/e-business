# 📘 Panduan Memulai (Untuk Tim)

Halo Tim! Ikuti langkah ini agar kamu bisa langsung kerja bareng Antigravity di laptop masing-masing:

## 1. Persiapan Awal
1.  **Install Node.js**: Download di [nodejs.org](https://nodejs.org/).
2.  **Install Git**: Download di [git-scm.com](https://git-scm.com/).
3.  **Buka Terminal**, arahkan ke folder tempat kamu ingin menyimpan proyek, lalu ketik:
    ```powershell
    git clone https://github.com/Hyfhi-M123/e-business.git
    ```
4.  Buka folder tersebut di VS Code.

## 2. Cara Kerja Bareng Antigravity
Cukup berikan perintah dalam bahasa Indonesia yang manusiawi. Contoh:
- *"Antigravity, tolong buatkan halaman login yang mewah ya."*
- *"Coba cek project_plan.md, sekarang tugas saya apa?"*
- *"Update dong kodingan saya ke GitHub."*

## 3. Aturan Main
- **JANGAN** hapus file `INSTRUKSI_ANTIGRAVITY.md` dan `LOG_AKTIVITAS.md`.
- Selalu suruh AG untuk **UPDATE LOG** setiap kali selesai melakukan perubahan penting.
