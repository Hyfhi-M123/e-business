Keamanan & Middleware ΓÇö Proteksi Rute Admin
Anggota 3 ΓÇö Supabase Auth + Next.js Middleware untuk autentikasi & otorisasi
 Alur Middleware Autentikasi
1
Request masuk ke /admin/* atau /dashboard/*
2
Next.js Middleware intercept request
3
Cek session token via Supabase Auth
 
Token valid Request diteruskan ke halaman
 
Token invalid Redirect ke /login
Supabase Project Setup
 API Keys diisolasi di .env.local
 Client connections via createClient()
 Row Level Security (RLS) diaktifkan
Auth Strategy
 Email/Password + Magic Link
 Session management otomatis
 JWT token untuk API calls
Route Protection
 middleware.ts di root project
 Matcher: /admin/*, /dashboard/*
 Redirect ke /login jika unauthorized
