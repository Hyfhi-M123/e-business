# 📊 Business Model Canvas & Analisis Kompetitor

## 1. Business Model Canvas (BMC)

| Segmen | Detail |
| :--- | :--- |
| **Customer Segments** | UI/UX Designer, Frontend Developer, Indie Hacker, Agency Web/Desain. |
| **Value Proposition** | Kurasi aset digital berkualitas tinggi (Tolak aset abal-abal), Pembelian 1-klik yang instan, Lisensi yang transparan. |
| **Channels** | X/Twitter (Komunitas Tech), Product Hunt, Dribbble, SEO Organic. |
| **Customer Relationship** | Support responsif, Refund Policy jelas, Komunitas Discord Eksklusif. |
| **Revenue Streams** | Potongan komisi dari kreator (10-15%), Biaya langganan bulanan "Lumina Pro" (akses unlimited untuk aset tertentu). |
| **Key Resources** | Platform Web (Next.js), Database (Supabase), Gateway Pembayaran. |
| **Key Activities** | Pemeliharaan platform, Peninjauan kualitas produk kreator (Kurasi), Marketing. |
| **Key Partners** | Kreator Digital Top Tier, Payment Gateway (Midtrans/Stripe), Layanan Hosting (Vercel). |
| **Cost Structure** | Server & Database, Biaya Gateway Pembayaran, Pemasaran/Iklan. |

## 2. Analisis Kompetitor (SWOT Singkat)

### Kompetitor 1: UI8.net
- **Kekuatan**: Katalog sangat besar dan beragam.
- **Kelemahan**: UI/UX website mulai terasa kuno, terlalu penuh (cluttered).
- **Peluang Kita**: Lumina bisa menawarkan antarmuka yang jauh lebih "clean" dan "breathable".

### Kompetitor 2: ThemeForest
- **Kekuatan**: Dukungan korporat besar (Envato), traffic masif.
- **Kelemahan**: Proses pendaftaran kreator rumit, potongan komisi sangat besar untuk kreator.
- **Peluang Kita**: Memberikan potongan komisi yang lebih adil bagi kreator untuk menarik talenta terbaik.
